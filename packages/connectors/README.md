# `@flue/connectors` (deprecated)

This package has been deprecated. Connectors are no longer distributed as
a separate npm package — install them with `flue add <name>` instead, which
fetches up-to-date instructions from `https://flueframework.com/cli/connectors/`
and pipes them to your AI coding agent.

For the previous Daytona connector:

```bash
flue add daytona | claude    # or opencode, codex, cursor-agent, etc.
```

Run `flue add` with no arguments to see the full list of available
connectors.
