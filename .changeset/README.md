# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the changesets repo](https://github.com/changesets/changesets).

## Adding a changeset

To add a changeset, run `pnpm changeset` in the root of the repository. You'll be prompted to select which
packages have changed, what kind of version bump to apply, and a summary of the change.

A changeset file will be created in this directory. Commit it along with your PR — when the PR is merged,
the automated release workflow will collect pending changesets, bump versions, update changelogs, and
publish to npm.
